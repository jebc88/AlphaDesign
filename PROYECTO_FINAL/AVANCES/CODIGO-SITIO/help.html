Ayuda de usuario

Sección de registrar usuario

Existen 3 tipos de usuarios los cuales se registran en el sistema:

Administrador: El cual posee absolutamente todos los permisos para administrar el sistema.

Normal: Este usuario únicamente puede crear maquinas virtuales.

Automatizado: Se encarga de las tareas automatizadas.

Sección usuarios registrados

Podemos observar los usuarios del sistema, además podemos desactivar el usuario (bloquearlo). Se nos brinda la accion de ver detalles y de eliminarlo de la base de datos.

 

Sección maquinas virtuales

Se observa el inventario de máquinas virtuales creadas, inluso podemos observar los detalles de cada una de estas.

 

Sección de configuración:

Nueva virtual: Podemos crear una nueva maquina.

Generar Respaldo de base de datos: Como se indica, permite generar el respaldo de la base de datos Unicenta de la maquina virtual creada.

Eliminar máquina: Realizamos la eliminacion de la maquina virtual seleccionada.

Restaurar máquina: Realizamos una restauracion de la maquina virtual solamente si esta fue eliminada en un tiempo menor de 1 mes.
